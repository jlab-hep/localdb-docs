* for YARR SW installation
    * g++ version 7.0 or higher
    * cmake3
    * gnuplot
    * epstopdf (TeXLive)
    * Ghostscript
* for Local DB
    * MongoDB
    * python3 & several modules
        * arguments
        * coloredlogs
        * Flask
        * Flask-PyMongo
        * Flask-HTTPAuth
        * Flask-Mail
        * pdf2image
        * Pillow
        * prettytable
        * pymongo
        * python-dateutil
        * PyYAML
        * pytz
        * plotly
        * matplotlib
        * numpy
        * requests
        * tzlocal
        * itkdb
